Pro:tkinter_img_to_threshold
============

Getting Started
-----
ubuntu:  
install numpy
```shell script
sudo pip3 install numpy
```
install opencv
```shell script
sudo pip3 install opencv-python
```
install Pillow
```shell script
sudo pip3 install Pillow
```
install tkinter(linux's python don't have tkinter)
```shell script
sudo apt-get install python3-tk
```

Prerequisities
-----------
* python3
* pip3
* ubuntu desktop version

Running the tests
--------
```shell script
python3 tkinter_img.py
```

Function
---
* 1.choose any picture  
* 2.adjust two parameter of cv2.adaptiveThreshold(function's name):  
  2.1 blockSize  
  2.2 C  
* 3.change picture to threshold
